# 📊 GitHub Profile Analyzer Pro

A beautiful, powerful GitHub profile analyzer with AI-powered insights, language analytics, and contribution scoring. **No backend required!**

## ✨ Features

- 🔍 **Instant Profile Analysis** - Enter any GitHub username
- 📈 **Detailed Statistics** - Repos, stars, forks, followers, and more
- 🎨 **Language Distribution** - Visual breakdown of programming languages
- 🏆 **Contribution Score** - Custom scoring algorithm (0-100)
- 💡 **Smart Suggestions** - Personalized improvement tips
- ⏱️ **Recent Activity** - Latest GitHub events in real-time
- 🌓 **Dark/Light Mode** - Toggle between themes
- 📱 **Responsive Design** - Works on all devices
- 🚀 **No Backend** - Runs entirely in your browser

## 🛠️ Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- GitHub REST API
- Chart.js for charts
- Font Awesome Icons
- Google Fonts (Inter)

## 📊 How It Works

1. Enter a GitHub username
2. The app fetches public data from GitHub API
3. Analyzes repositories, stars, forks, languages
4. Displays comprehensive profile insights
